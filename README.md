# Smart Retail ERP System

A fully automated retail inventory and supplier management system built on Microsoft Power Platform (Dataverse, Canvas App, Model-Driven App, Power Automate).

## Contents
- `SmartRetailERP_managed.zip` — Exported managed solution
- `Project_Report.docx` — Full project report (ERD, architecture, user flow, screenshots)
- `Presentation.pptx` — Project presentation slides

## Features
- Canvas App for product browsing and sales logging
- Model-Driven App with Business Process Flow (Pending → Approved → Delivered)
- 3 Power Automate flows:
  - Low Stock Alert (dynamic supplier email)
  - Order Approved Notify (dynamic supplier email)
  - Daily Low-Stock Digest (scheduled)
